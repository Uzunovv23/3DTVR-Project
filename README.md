Интерактивно образователно приложение с добавена реалност (AR), създадено с Unity и Vuforia Engine. Този проект демонстрира проследяване на изображения (Image Tracking) и визуализация на 3D модели на небесни тела върху физически маркери.

Характеристики на проекта:

Множествено разпознаване на маркери: Едновременно или последователно засичане на 3 различни изображения (Слънце, Земя, Луна).

3D Визуализация: Рендериране на 3D сфери с реалистични текстури на небесните тела, закачени за физическите маркери.

Интерактивна логика (C#): Непрекъснато орбитално въртене на планетите. Потребителско взаимодействие – при клик върху планетата се визуализира интересен факт на екрана.

Оптимизиран код: Изчистена йерархия и използване на TextMeshPro за кристално ясен потребителски интерфейс.

Използвани технологии:

Среда за разработка: Unity 2022.3 LTS

AR SDK: Vuforia Engine

Език за скриптиране: C#

Ако искате да стартирате проекта локално на вашия компютър:

Клонирайте хранилището:
git clone https://github.com/Uzunovv23/3DTVR-Project.git

Отворете проекта в Unity:
Отворете Unity Hub.
Изберете Open -> Add project from disk и посочете клонираната папка.
Забележка: Проектът изисква инсталиран Vuforia Engine пакет.

Стартиране:
Отворете сцената SampleScene (или както сте я кръстили).
Натиснете бутона Play.
Насочете уеб камерата към съответните картинки-маркери (намират се в папка Assets), за да видите магията на добавената реалност.
